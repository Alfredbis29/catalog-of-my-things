## Get Started
```
To get a clone of the project, run `git clone git@github.com:https://github.com/zmekonnen251/catalog-of-my-things.git`
```
```
cd /catalog-of-my-things
```

### Available Scripts

In the project directory, you can run:

**`run ruby app.rb`**
